# Documentation Agent: verify & author docs

Build and roll out a Documentation Agent that runs after the Implementation agent finishes. The agent will ensure that implemented work is captured in the project documentation and decision logs: it will find relevant docs/decisions, create new documents or ADRs when missing, and append task-level documentation and links back to the originating tasks.
